# 説教でしゅ！！
<div class="novel_view" id="novel_honbun">
 <p id="L1">
  可愛らしい幼女がウロウロと私の部屋を右へ左と移動しています。私から見てもとても可愛らしいんですが眉間にシワを寄せ怒っています。
 </p>
 <p id="L2">
  折角の可愛い顔が台無しです。
 </p>
 <p id="L3">
 </p>
 <p id="L4">
  お洋服はお母さんに買ってもらったものでワンピース（ネイビーに黒の水玉）レギンス（黒）にスリッパ（黒、猫の絵柄入り）。
 </p>
 <p id="L5">
  髪はおろしていますね。頭の左側にはネコさんヘアピンが存在感をはなっています。
 </p>
 <p id="L6">
  最近暑くなってきたので少し涼しい感じになってきています。今日のマスターも可愛いです！
 </p>
 <p id="L7">
  <br/>
 </p>
 <p id="L8">
  「トリャ！　聞いてましゅか！」
 </p>
 <p id="L9">
  <br/>
 </p>
 <p id="L10">
  「は、はい」
 </p>
 <p id="L11">
  <br/>
 </p>
 <p id="L12">
  マスターは怒っています。なんでも來実さん、珠理亜さん、彩葉さんに対しての対応が間違っていたらしいのです。
 </p>
 <p id="L13">
  <br/>
 </p>
 <p id="L14">
  「いいでしゅか？　トリャは今、男なのでしゅ！　しかも頭よくて、カッコイイのでしゅ。しょんな男が女の子に甘い言葉を囁くとどうなりましゅか？」
 </p>
 <p id="L15">
  <br/>
 </p>
 <p id="L16">
  「はい！　マスター質問です。甘い言葉ってなんですか？」
 </p>
 <p id="L17">
  <br/>
 </p>
 <p id="L18">
  「えっとっでしゅね……
  <ruby>
   <rb>
    やさちく
   </rb>
   <rp>
    (
   </rp>
   <rt>
    優しく
   </rt>
   <rp>
    )
   </rp>
  </ruby>
  その人が喜ぶ様な言葉を掛けることでしゅ」
 </p>
 <p id="L19">
  <br/>
 </p>
 <p id="L20">
  はて？　優しい言葉を掛けて、更に喜ぶ言葉を掛けたら嬉しいんじゃないんですかね？　マスターの意図が読めないので素直に思ったことを答えます。
 </p>
 <p id="L21">
  <br/>
 </p>
 <p id="L22">
  「喜びます」
 </p>
 <p id="L23">
  <br/>
 </p>
 <p id="L24">
  「そうでしゅ！　喜ぶのでしゅ。そしてどうなるでしゅか？」
 </p>
 <p id="L25">
  <br/>
 </p>
 <p id="L26">
  どうなる？　考える……私の経験は非常に少ない。データーとして存在していたときに閲覧していた情報を思い出す。
 </p>
 <p id="L27">
  <br/>
 </p>
 <p id="L28">
  外に出たら見たいなって思っていた風景、可愛い動物、ファッション、後は、ん～と、そうお兄ちゃん、そして家族ってなんだろうって……
 </p>
 <p id="L29">
  <br/>
 </p>
 <p id="L30">
  マスターの求める答えを導き出すにはあんまり役に立ちそうにないです。喜ぶ……嬉しい……あっ！
 </p>
 <p id="L31">
  <br/>
 </p>
 <p id="L32">
  「その人のこと好きになります！」
 </p>
 <p id="L33">
  <br/>
 </p>
 <p id="L34">
  「そうでしゅ！　しゅきになるのでしゅ！」
 </p>
 <p id="L35">
  <br/>
 </p>
 <p id="L36">
  マスターの顔が喜びで満ち溢れています。正解のようです！　嬉しいです。
 </p>
 <p id="L37">
  <br/>
 </p>
 <p id="L38">
  「トリャが女の子に
  <ruby>
   <rb>
    やさちく
   </rb>
   <rp>
    (
   </rp>
   <rt>
    やさしく
   </rt>
   <rp>
    )
   </rp>
  </ruby>
  しゅる。女の子はトリャがしゅきになる。はい、どうなりましゅ？」
 </p>
 <p id="L39">
  <br/>
 </p>
 <p id="L40">
  「どうって好きになってもらって嬉しいです」
 </p>
 <p id="L41">
  <br/>
 </p>
 <p id="L42">
  「ま、まあそうなんでしゅけど。お前は
  <ruby>
   <rb>
    たくしゃん
   </rb>
   <rp>
    (
   </rp>
   <rt>
    沢山
   </rt>
   <rp>
    )
   </rp>
  </ruby>
  の人にそのしゅきを与えているのでしゅ。それは身を滅ぼす行為なんでしゅ」
 </p>
 <p id="L43">
  <br/>
 </p>
 <p id="L44">
  ？　なんで好きになってもらったら身を滅ぼすのでしょうか？　これは聞いてみないといけません。私は手を上げマスターに質問します。
 </p>
 <p id="L45">
  ですがマスターは若干半目になっていて頭をフラフラさせています。恐らく頭を酷使して眠たいんだと思います。
 </p>
 <p id="L46">
  なんでも２桁の掛け算を筆算している辺りから猛烈に眠気が襲ってくるそうです。
 </p>
 <p id="L47">
  昔は３桁まで暗算でいけたらしいのでショックを受けてました。
 </p>
 <p id="L48">
  でもフラフラして半目のマスターは可愛いです。
 </p>
 <p id="L49">
  <br/>
 </p>
 <p id="L50">
  「すいません、なんで身を滅ぼすんですか？」
 </p>
 <p id="L51">
  <br/>
 </p>
 <p id="L52">
  「
  <ruby>
   <rb>
    たくしゃん
   </rb>
   <rp>
    (
   </rp>
   <rt>
    沢山
   </rt>
   <rp>
    )
   </rp>
  </ruby>
  の人にしゅきになってもらうと、しょの人たちがしゅきを争いトリャを傷ちゅけ、本人たちも傷ちゅくことになるのでしゅ」
 </p>
 <p id="L53">
  <br/>
 </p>
 <p id="L54">
  「傷付くんですか？」
 </p>
 <p id="L55">
  <br/>
 </p>
 <p id="L56">
  「しょう、今くりゅみ、珠理亜はトリャのことしゅきでしゅ。
  <ruby>
   <rb>
    いりょは
   </rb>
   <rp>
    (
   </rp>
   <rt>
    彩葉
   </rt>
   <rp>
    )
   </rp>
  </ruby>
  はギリギリ
  <ruby>
   <rb>
    シェーフ
   </rb>
   <rp>
    (
   </rp>
   <rt>
    セーフ
   </rt>
   <rp>
    )
   </rp>
  </ruby>
  でしゅかね。この子たちにしゅきを与え
  <ruby>
   <rb>
    しゅぎ
   </rb>
   <rp>
    (
   </rp>
   <rt>
    すぎ
   </rt>
   <rp>
    )
   </rp>
  </ruby>
  なのでしゅ」
 </p>
 <p id="L57">
  <br/>
 </p>
 <p id="L58">
  …………思考中です。好きを与えすぎると傷付く……！？　分かりました！
 </p>
 <p id="L59">
  <br/>
 </p>
 <p id="L60">
  「好きを偏らせず均等にしろってことですね！　分け隔てなく好きを配ったら皆が嬉しくなって喜びますもんね！！　片寄ったら不公平ですし、お花だって水のあげすぎはよくありませんからね。マスター流石です！」
 </p>
 <p id="L61">
  <br/>
 </p>
 <p id="L62">
  「ん？　そりぇ、合ってるのでしゅ？」
 </p>
 <p id="L63">
  <br/>
 </p>
 <p id="L64">
  マスターは目を擦りながらムニュムニュいい始めました。相当眠いようですね、そしてものすごーーーーく、可愛いです♪
 </p>
 <p id="L65">
  <br/>
 </p>
 <p id="L66">
  「マスター下で寝ましょう。ここで寝ると私がお母さんに殺されます」
 </p>
 <p id="L67">
  <br/>
 </p>
 <p id="L68">
  「んー」
 </p>
 <p id="L69">
  <br/>
 </p>
 <p id="L70">
  返事も可愛いです！　キュンキュンします。私は半目で目をグリグリ擦るマスターと手を繋ぎ階段を降りて行きマスターのベットまで連れて行くとそっと横に寝かせます。
 </p>
 <p id="L71">
  <br/>
 </p>
 <p id="L72">
  アンドロイドなんで寝息はたてませんが横になるとすぐにスースーと寝てしまいました。
 </p>
 <p id="L73">
  グッスリ眠ったマスターの寝顔を見て私は頬をツンツンしてみます。
 </p>
 <p id="L74">
  柔らかい感触が指先に伝わりなんだか幸せです。
 </p>
 <p id="L75">
  更にプニプニと突っつくと小さい手で私の手を払いのけてきます。
 </p>
 <p id="L76">
  <br/>
 </p>
 <p id="L77">
  たまりません！！　これが幸せってやつなんでしょうね。
 </p>
 <p id="L78">
  <br/>
 </p>
 <p id="L79">
  基本アンドロイドは決められた時刻と時間しか寝ません。寝るといってもオートで行うボディーチェックをしているのです。
 </p>
 <p id="L80">
  でもマスターは他のアンドロイドと違い、いつでも寝ますし時間もまばらだったりします。
 </p>
 <p id="L81">
  別に体に異常はないようなのでマスターと私が入れ替わったことによるものなのかもしれません。
 </p>
 <p id="L82">
  <br/>
 </p>
 <p id="L83">
  入れ替わったといえば元に戻る方法を探してはいるんですが、正直行き詰まってます。
 </p>
 <p id="L84">
  と言いますか全然分かりません。それはマスターも理解しているようで、そんなには急かされません。
 </p>
 <p id="L85">
  どちらかと言うとアンドロイドの知識を身に付けて欲しいと言われてます。
 </p>
 <p id="L86">
  なので差し詰めメモリーの増設を目標に勉強に励んでいるんです。少しは賢くなりましたけどまだまだです。マスターを分解して戻せなくなったら大変ですからね。
 </p>
 <p id="L87">
  <br/>
 </p>
 <p id="L88">
  寝ていても可愛らしいマスターの頭を撫でるとムニムニいいながら頭を振ってイヤイヤします。
 </p>
 <p id="L89">
  <br/>
 </p>
 <p id="L90">
  本当に可愛いです。思わず顔がニヤケます。
 </p>
 <p id="L91">
  <br/>
 </p>
 <p id="L92">
  ……マスターは元に戻りたいんですよね？　私はどうなんでしょう？　元々心春になる予定だった私。でも気が付けばこのマスターの元の体。
 </p>
 <p id="L93">
  正直心春の体に愛着が無いというかあっちが本当の体だって実感がないんですよね。戻りたいじゃなくてこのままマスターと一緒にいたいなぁって感じです。
 </p>
 <p id="L94">
  そんなこと言えばマスターが悲しむでしょうけど……う～ん、なんなでしょう？　私が初めからこの体なので余計な考えを持ってしまうんでしょうか。
 </p>
 <p id="L95">
  <br/>
 </p>
 <p id="L96">
  「ムニュ～」
 </p>
 <p id="L97">
  <br/>
 </p>
 <p id="L98">
  おぉ！　マスター可愛いです！　何て言うか愛おしいです！
 </p>
 <p id="L99">
  愛おしさ頂点の私はそっとマスターの頬に右手を添えるとゆっくりと唇をマスターの額に近付けます。
 </p>
 <p id="L100">
  <br/>
 </p>
 <p id="L101">

 </p>
 <p id="L102">
  <br/>
 </p>
 <p id="L103">
  頭が軋みます……
 </p>
 <p id="L104">
  <br/>
 </p>
 <p id="L105">
  「トラ……声出したら潰す」
 </p>
 <p id="L106">
  <br/>
 </p>
 <p id="L107">
  「うっ！？」
 </p>
 <p id="L108">
  <br/>
 </p>
 <p id="L109">

 </p>
 <p id="L110">
  <br/>
 </p>

 </p>
 <p id="L112">

 </p>
 <p id="L114">
  <br/>
 </p>
 <p id="L115">

 </p>
 <p id="L116">

 </p>
 <p id="L118">
  <br/>
 </p>
 <p id="L119">

 </p>
 <p id="L120">
  <br/>
 </p>
 <p id="L121">
  「最近ちょっとまともになったと思ったけど、２人っきりにしたらこれだから油断も隙もありゃしないわ。やっぱり家では接触を制限しないといけないわね」
 </p>
 <p id="L122">
  <br/>
 </p>

 <p id="L124">

 <p id="L127">
